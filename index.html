ACADEMIC LITERACIES MODEL

A framework for understanding literacy practices in academic contexts

Academic literacies foreground meaning-making, identity, power, and authority. In academic writing, moving beyond skills and socialisation models.

The Models of Academic Literacy

Study Skills Model

  • Assumes literacies are transferable skills
  • Grammar, punctuation, sentence structure
  • Assumes skills apply across all contexts

Academic Socialisation Model

  • Acculturation into disciplinary discourses
  • Learning genres and conventions
  • Assumes stability in disciplinary norms

Academic Literacies Model

  • Focus on meaning-making, identity, power
  • Recognises literacy practices as situated
  • Embraces complexity and variability

Disciplinary Literacy

Disciplinary literacy emphasises the specialised knowledge and abilities used by experts within each discipline.

Key distinctions from content area literacy:

Discipline-Specific

Focuses on discipline-specific practices

Unique Tools

Uses unique tools of each discipline

Fundamental Differences

Recognises fundamental differences between disciplines

Varied Purposes

Addresses varied purposes, genres, and communication traditions

Research approaches:

  • Expert–novice comparisons
  • Functional linguistics analysis
  • Think-aloud protocols
